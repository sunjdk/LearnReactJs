开发环境搭建

1.引入.js文件来使用React

2.通过脚手架工具来编码		grunt	gulp	webpack

官方脚手架工具 create-react-app

步骤1：安装nodejs 

步骤2：node -v 确认环境版本

步骤3：npm install -g create-react-app

步骤4：mkdir myreact

步骤5：cd myreact

步骤6：create-react-app todolist

步骤7：cd todolist

步骤8：yarn start   启动项目
